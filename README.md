# Range Finder

<a id="english"></a>

Range Finder is an iOS app that uses laser sensors to measure the distance between your phone and a target object.

## Features

- Real-time distance measurement using laser sensors
- Simple and intuitive user interface

## Installation

1. Clone this repository
2. Open `range-finder.xcodeproj` in Xcode
3. Build and run the project on your iOS device

## Usage

1. Launch the Range Finder app
2. Point your device at the target object
3. The distance will be displayed on the screen
